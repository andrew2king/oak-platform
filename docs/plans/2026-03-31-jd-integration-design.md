> 版本：v1.0

> 状态：已批准

## 一、项目背景

shop-web商城前台需要对接京东商品平台，实现一件代发模式销售。同时美化首页视觉效果，提升用户体验。

---

## 二、商业模式

**一件代发模式**

用户在Oak平台下单 → 系统自动向京东下单 → 京东直发给用户 → 用户在Oak查看物流

**优势：**
- 用户体验好（不跳转，统一收银台）
- 利润空间可控（加价销售）
- 无需仓储物流
- 品质有保障（京东商品）

```
│  │ MockAdapter │  │ JDAdapter   │  │ ExtAdapter  │         │

└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    后端服务 (Node.js/Go)                     │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │ 商品同步    │  │ 订单转发    │  │ 物流追踪    │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    京东开放平台 API                           │
│  - jd.union.open.goods.jingfen.query (精选商品查询)          │
│  - jd.union.open.order.row.query (订单查询)                  │
│  - jd.union.open.order.validate (订单校验)                   │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 核心API模块

| 模块 | 功能 | 京东API |
|------|------|---------|
| 商品同步 | 定时拉取京东商品库 | jd.union.open.goods.jingfen.query |
| 实时库存 | 查询商品库存状态 | jd.union.open.goods.promotiongoodsinfo.query |
| 订单创建 | 用户下单后向京东下单 | 通过推广链接跳转/PI下单 |
| 物流追踪 | 同步物流信息 | jd.union.open.order.row.query |
| 售后处理 | 退换货对接 | 京东售后API |

### 3.3 数据流程

```
用户浏览商品
    │
    ▼
展示本地缓存的京东商品（定时同步）
    │
    ▼
用户下单支付
    │
    ▼
系统生成推广链接/向京东下单
    │
    ▼
京东发货 → 物流信息回传
    │
    ▼
用户在Oak平台查看物流
```

## 六、技术要点

### 6.1 京东联盟API接入

```javascript
// 京东联盟API配置
const JD_CONFIG = {
  appKey: 'YOUR_APP_KEY',
  appSecret: 'YOUR_APP_SECRET',
  unionId: 'YOUR_UNION_ID',
  positionId: 'YOUR_POSITION_ID', // 推广位ID
  apiGateway: 'https://api.jd.com/routerjson'
}

// 商品查询示例
const goodsList = await jdApi.request('jd.union.open.goods.jingfen.query', {
  eliteId: 1, // 频道ID：1-好券商品, 2-精选卖品
  pageIndex: 1,
  pageSize: 20
})
```

### 6.2 商品数据映射

```typescript
// 京东商品 → Oak商品结构
function mapJDProduct(jdGoods: JDGoods): Product {
  return {
    id: jdGoods.skuId,
    name: jdGoods.goodsName,
    price: jdGoods.priceInfo.price,
    originalPrice: jdGoods.priceInfo.originalPrice,
    image: jdGoods.imageInfo.imageList[0].url,
    images: jdGoods.imageInfo.imageList.map(i => i.url),
    sales: jdGoods.inOrderCount30Days,
    categoryName: jdGoods.categoryInfo.cname,
    brandName: jdGoods.brandName,
    // 加价销售策略
    salePrice: jdGoods.priceInfo.price * 1.15 // 加价15%
  }
}
```

## 七、风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|---------|
| 京东API限流 | 商品加载慢 | 本地缓存 + 降级策略 |
| 商品下架 | 用户下单失败 | 实时库存校验 + 友好提示 |
| 价格变动 | 利润波动 | 定时同步 + 价格监控 |
| 物流延迟 | 用户体验差 | 物流追踪 + 客服介入 |
